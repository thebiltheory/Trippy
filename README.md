# Trippy

## Trip Sorter

Try the [demo](http://trippy.thebiltheory.com)

Install Yarn (if not already have)
```Bash
$ npm install -g yarn
```
Install dependencies
```Bash
$ yarn
```

Launch dev and local server
```Bash
$ yarn watch
```

Build
```Bash
$ yarn build
```

Run Tests
```Bash
$ yarn test
```
